## Story

As a developer, I want a curated set of high-quality Agent Skills so that my AI assistant follows best practices without me writing rules from scratch.

## Acceptance Criteria

1. 23 skills across 8 categories (general, typescript, git, ui, backend, testing, lua, golang)
2. Each skill has clear instructions, checklists, and examples
3. Skills are project-agnostic (no hardcoded project references)
4. Skills are maintained and versioned
