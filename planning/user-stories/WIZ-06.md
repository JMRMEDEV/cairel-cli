## Story

As a new user, I want to see an interactive menu when I run cairel without arguments so that I can discover available commands.

## Acceptance Criteria

1. Running `cairel` (no args) shows interactive menu
2. Menu lists: init, bootstrap, update, validate, list, help
3. Selecting an option runs the corresponding command
