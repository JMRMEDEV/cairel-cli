# GEN-07: Multi-platform simultaneous output

## Metadata

| Field | Value |
|-------|-------|

| Priority | P1 |
| Scope | Done |
| Domain | generation |
| Subprojects | cli |

## Story

As a developer using multiple AI tools, I want to select multiple platforms simultaneously so that all my tools get configured in one run.

## Acceptance Criteria

1. Platform selection is multi-select (not single-select)
2. Skills generated to all selected platform directories
3. Agent JSON generated for platforms that use it (Kiro, Amazon Q)
