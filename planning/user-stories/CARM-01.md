## Story

As a developer, I want a central registry for AI rules and skills so that I can discover and install community-contributed best practices.

## Acceptance Criteria

1. Registry backend with search, browse, and download capabilities
2. Package format with metadata, versioning, and compatibility info
3. Namespace system (@cairel/*, @community/*, @username/*)
4. Package verification and integrity checks
