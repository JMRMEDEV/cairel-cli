# Cairel Curated Skills

This directory contains standardized, project-agnostic Agent Skills following the [agentskills.io](https://agentskills.io) open specification.

## Format

Each skill is a directory containing a `SKILL.md` file:

```
skill-name/
├── SKILL.md           # Required: metadata + instructions
└── references/        # Optional: additional documentation
```

### Frontmatter

Skills use the standard Agent Skills frontmatter with cairel-specific metadata:

```yaml
---
name: skill-name
description: What this skill does and when to use it. Max 1024 chars.
metadata:
  cairel-category: general
  cairel-version: "1.0.0"
  cairel-tags:
    - tag1
    - tag2
  cairel-always-include: true          # Optional: always selected
  cairel-conditions:                   # Optional: wizard selection logic
    languages:
      - typescript
    frameworks:
      - react
    project-types:
      - ui
    ui-library:
      - chakra-ui
    linter:
      - eslint
    versioning-strategy:
      - semantic
    requires-git: true
    requires-env-vars: true
---
```

### How It Works

- **AI tools** (Kiro, Copilot, Claude Code) discover skills via `name` and `description`
- **Cairel** uses `metadata.cairel-*` fields for wizard-driven conditional selection
- Both work independently — no conflict

## Platform Compatibility

Skills generated by cairel work natively with:
- **Kiro** → `.kiro/skills/`
- **GitHub Copilot** → `.github/skills/` or `.agents/skills/`
- **Claude Code** → `.claude/skills/`

## Validation

```bash
# Validate with cairel
cairel validate

# Validate with official skills-ref tool
skills-ref validate ./skill-name
```
